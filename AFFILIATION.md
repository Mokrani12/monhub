# Guide d'affiliation — MonHub

Pour encaisser de l'argent, remplacez les URLs génériques de `assets/links.js`
par VOS liens d'affiliation. Inscriptions gratuites :

| Programme | Inscription gratuite | À mettre dans links.js |
|-----------|----------------------|------------------------|
| Amazon Partenaires | https://affiliate-program.amazon.fr/ | `https://www.amazon.fr/?tag=VOTRE_ID` |
| Skillshare | https://www.skillshare.com/affiliates | `https://skillshare.eqcm.net/VOTRE_ID` |
| Booking.com Partners | https://www.booking.com/affiliate-program/ | `https://www.booking.com/?aid=VOTRE_ID` |
| AliExpress Portal | https://portals.aliexpress.com/ | lien de tracking fourni |
| TopCashback | https://www.topcashback.fr/affiliate | lien de tracking fourni |
| Ko-fi (tips) | https://ko-fi.com/ | `https://ko-fi.com/VOTRE_COMPTE` |

## AdSense (publicité)
1. Créez un compte gratuit sur https://www.google.com/adsense/ (demande un peu de trafic).
2. Dans `index.html`, décommentez le bloc `<script async ... adsbygoogle.js>`
   et remplacez `ca-pub-XXXXXXXXXXXXXXX` par votre ID.
3. Ajoutez des emplacements publicitaires dans les pages outils si vous voulez.

## Trafic (gratuit)
- Partagez vos outils (QR, budget, captions, mots de passe) sur réseaux, forums, Reddit, groupes Facebook.
- Publiez le lien bio-link dans votre bio Instagram/TikTok/YouTube.
- Soumettez le site sur des annuaires gratuits.
